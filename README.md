# LibreShare

## Introduction

LibreShare is a small web based file synchronization like Nextcloud.

## Instructions

Build requirements: `Java 17+ JDK, Docker, PostgreSQL`

Commands:

```java
./gradlew clean build

java -jar ./src/web/build/libs/web-0.0.1-all.jar ./config/application.json
```
